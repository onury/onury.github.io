re.js

Javascript RegExp API for Humans! ...and for Node and Browser.

.. each() instead of RegExp#exec() and more..

Installation

Install via NPM:

npm i re.js --save

Features:

  • JS has a weird implementation of Regular Expressions. Some utility methods are only on String prototype, and not on RegExp.
  • Automatically resets .lastIndex when needed. A) when you call the re() constructor. B) when re call is done. If you don't know why/what this is; read this, this, this or don't worry since this is made to produce the expected behavior.
